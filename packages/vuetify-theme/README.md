# `@deltares/vuetify-theme`

The Deltares theme for Vuetify.

## Install

```
npm install @deltares/vuetify-theme
```

## Usage

```js
import Vue from 'vue';
import Vuetify from 'vuetify/lib';
import themes from '@deltares/vuetify-theme'

Vue.use(Vuetify);

export default new Vuetify({
  theme: {
    themes
  },
});
```
